# MIXTEAM SPORTS Website

Static marketing website for MIXTEAM SPORTS, a Bay Area youth sports agency helping athletes prepare for college opportunities.

## Files

- `index.html` - page structure and copy
- `styles.css` - responsive visual system
- `script.js` - mobile navigation and intake form response
- `assets/hero-athletes.png` - generated hero image
- `assets/concept-reference.png` - design concept used as implementation reference

## Hosting
